# Github Configuration Guide

Authentication through [Github ↗](https://github.com/) enables you to sign in to the platform using your Github account. For detailed guidance on how to implement this, please follow the official links from the provider..

- [Documentation](https://developer.github.com/apps/building-oauth-apps/authorizing-oauth-apps)
- [Configuration](https://github.com/settings/apps)
- [OAuth App](https://github.com/settings/developers)

#### Github OAuth App Settings

Create a new [OAuth App](https://github.com/settings/developers) by filling out the form with the following settings:

```bash
"Application name": "Feature Flags by Basestack"
"Homepage URL": "https://your-basestack-instance.com"
"Authorization callback URL": "https://your-basestack-instance.com"
```

#### Self-Hosting Environment Variables

| Property               | Required |
| ---------------------- | -------- |
| `GITHUB_CLIENT_ID`     | true     |
| `GITHUB_CLIENT_SECRET` | true     |
